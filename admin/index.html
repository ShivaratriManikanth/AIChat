Embed on Your Website

Add this single line of code to any HTML, PHP, WordPress, or Laravel website.

Universal Script Tag (Supports: English, Hindi, Telugu)

<script src="/widget/chatbot.js" data-server=""></script>

WordPress (functions.php)

function add_chatbot_widget() { echo '<script src="/widget/chatbot.js" data-server=""></script>'; } add_action('wp_footer', 'add_chatbot_widget');

Laravel (Blade Template)

{{-- Add before </body> in layout --}} <script src="/widget/chatbot.js" data-server=""></script>
